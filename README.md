# Clarity-AI

An AI-powered decision-making assistant that helps users make informed choices by analyzing situations and providing clear recommendations using local AI models.

## Overview

Clarity-AI is a web application that leverages **Ollama** and **Llama3** to provide intelligent decision recommendations. Users can describe a decision they're facing, and the AI analyzes it to provide a clear conclusion and reasoning.

## Features

✨ **AI-Powered Recommendations** - Uses Llama3 model via Ollama to analyze decisions
📝 **Decision History** - Keeps track of all decisions and recommendations
🗑️ **Delete History** - Remove past decisions from your history
⚡ **Real-time Processing** - Get immediate AI analysis
🎯 **Clear Output** - Structured recommendations with reasoning

## Tech Stack

- **Backend**: Python (44.8%) - Flask web framework
- **Frontend**: HTML (20.9%) - User interface templates
- **Styling**: CSS (34.3%) - Responsive design
- **Database**: SQLite - Local data persistence
- **AI Engine**: Ollama with Llama3 model

## Prerequisites

Before you begin, ensure you have the following installed:

- Python 3.8 or higher
- [Ollama](https://ollama.ai) - Local LLM runtime
- Llama3 model (download via Ollama)
